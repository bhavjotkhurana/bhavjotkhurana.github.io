Initially written to help give my students extra practice questions, this project is a curated collection of SAT-style math questions written and formatted in LaTeX, covering all four major SAT math domains:

- Algebra  
- Geometry & Trigonometry  
- Problem Solving & Data Analysis  
- Advanced Math

The repository includes both questions and full solutions, designed for self-study and eventual integration with a digital scrolling practice app (QScroll).

## Features

- Cleanly formatted PDF practice sets for offline or printable use  
- Fully worked solutions, written in LaTeX for clarity and consistency  
- Structured for integration into QScroll, a React-based platform for mobile-first SAT prep

## Tech Stack

- LaTeX for typesetting  
- Git/GitHub for version control and sharing  
- QScroll (planned) for interactive delivery
