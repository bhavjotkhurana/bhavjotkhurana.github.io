This web version brings the core qScroll experience to more devices and lays the foundation for scalable, personalized test prep—accessible from anywhere.

---

## Purpose

Standardized testing is predictable. Most prep platforms are not. The goal of qScroll is to build something different—**a stripped-down practice environment focused on deliberate repetition, adaptive feedback, and data collection** that actually informs what students should do next.

The browser version:
- Removes distractions (no progress bars, dashboards, or timers—just questions)
- Enables instant use (no download or install required)
- Provides a starting point for integrating backend analytics and personalized review

---

## Features

- Vertical scrolling interface: swipe or scroll to progress through questions
- Click-to-select answers with instant correctness feedback
- Visual indicators (green/red borders) for correct/incorrect responses
- Responsive design optimized for mobile and desktop browsers
- Keyboard-accessible for desktop users

---

## Tech Stack

- **React** for dynamic rendering (planned/actual)
- **HTML/CSS/JS** for initial prototype versions
- Support for:
  - Question tagging and filtering
  - Score tracking and local storage
  - Enabling authorization

## Broader Context

This app is part of a broader effort to rethink what effective, scalable test prep can look like. The end goal is not just to serve questions, but to build **adaptive educational tools that surface patterns, correct misconceptions, and respect students’ time**.

The web version complements the iOS version and is better suited for:
- Students using shared or low-storage devices
- Cross-platform access (school computers, Chromebooks, etc.)
- Future classroom integrations
